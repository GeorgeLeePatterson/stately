# @stately/schema-types

> Core type definitions for Stately schema nodes

This package contains TypeScript type definitions that are shared between:
- `@stately/codegen` - Build-time schema generation
- `@stately/ui` - Runtime UI components

## Installation

This package is typically installed as a dependency of other `@stately/*` packages and doesn't need to be installed directly.

```bash
# If you need to install it directly
pnpm add @stately/schema-types
```
